# 🔐 Cyber Threat Intelligence (CTI) Dashboard

A web-based CTI dashboard to analyze IP intelligence and support cyber investigations.

## 🚀 Features
- IP Lookup & Intelligence
- User Authentication (Login/Signup)
- Clean Dark UI Dashboard
- Real-time Results Display

## 🛠 Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, Bootstrap

## 🕵️ Use Case
- Cybercrime Investigation
- Threat Hunting
- SOC Analysis
- Digital Forensics

## ⚙️ Run Locally
```bash
pip install -r requirements.txt
python app.py
